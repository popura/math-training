\section{信号}
  信号とは，あるシステムの状態や振る舞いに関する情報を伝えるものである．
  信号は，数学的には，1つ以上の独立変数に関する関数（あるいは数列）として表される．
  次の信号は(a)から(d)のうちどれに該当するか．
  当てはまるものをすべて選べ．
  \begin{enumerate}[label=(\roman*)]
    \item 周期信号
    \item 因果性信号
    \item 連続信号
    \item 右側信号
    \item 左側信号
    \item 有限エネルギー信号
  \end{enumerate}
